# em_segLib: library of functions for segmentation operations

## Installation
`conda install --yes --file requirements.txt` or `pip install -r requirements.txt`

`python setup.py install`

## Package Functions

1. Segmentation
  - core (seg_core)
  - transformation (seg_dist)
  - evaluation (seg_eval)
  - malis weight (seg_malis)
  - utility (seg_util)

2. Affinity 
  - utility aff_util)
